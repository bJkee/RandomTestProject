сгенерировать миллион случайных цифр в диапазоне 0-1. Для сгенерированных чисел найти:
1. Количество [0], и количество [1]
2. Количество последовательностей из двух цифр (00, 01, 10, 11).
3. Количество последовательностей из трех цифр (000, 001, 010, и т.д.)

Из десяти сгенерированных цифр, нулей - 7, единиц 3. В процентах, соответственно, 70%/30%.

Пункт №3 - количество последовательностей из трех цифр [000], [001], [010], [011], [100], [101], [110], [111]

Пункт №4 - каждый из предыдущих пунктов выразить в виде процентного соотношения (выражено в виде диаграммы)
